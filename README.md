# SkillBridge AI

**Intelligent Resume–Job Matching with Skill Gap Analysis**

## Overview
SkillBridge AI predicts resume–job fit, identifies missing skills, and generates personalized learning recommendations using classical ML, RAG, and agent-based reasoning.

## Features
- Job-fit prediction
- Skill-gap analysis
- Grounded learning recommendations
- Full-stack demo (FastAPI + Streamlit)
